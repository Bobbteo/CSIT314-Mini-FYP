# CSIT314-Mini-FYP

## Run the website (Windows / PowerShell)

### 1) Install dependencies

```powershell
.\.venv\Scripts\python -m pip install -r requirements.txt
```

### 2) Start the app

```powershell
.\.venv\Scripts\python fundraising_system\app.py
```

### 3) Open in browser

Go to `http://127.0.0.1:5000`.

### Stop the server

Press `CTRL + C` in the terminal that’s running Flask.
